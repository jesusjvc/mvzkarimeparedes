# Genesis Base - EsBrillante

GitHub project link: https://github.com/EmyRub/genesis_base_sitio.


## Instrucciones de Instalación
1. Sube la carpeta del tema Genesis Sample a través de FTP a tu directorio wp-content/themes/. (El tema principal Genesis también debe estar en el directorio wp-content/themes/).
2. Accede a tu panel de administración de WordPress y selecciona "Apariencia".
3. Activa el tema Genesis Base - EsBrillante.
En el panel de administración de WordPress, ve a Genesis > Configuración del Tema y ajústalas según tus preferencias.
